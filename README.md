# AIN Helper

[![Version](https://img.shields.io/badge/version-1.1.3-blue.svg)](https://github.com/twiceKim/AIN_Helper/releases)

전투분석을 위한 DPS 미터기

네트워크 패킷을 캡처하여 게임 내 데미지 정보를 실시간으로 분석하고 오버레이 UI로 표시합니다.

> 해당 프로그램은 운영측의 요청, 패킷 암호화 등의 조치, 공식적인 사용중단 언급이 있다면 중지 및 비공개 상태로 전환됩니다.

## 주요 기능

### DPS 측정
- **실시간 DPS 계산**: `누적 데미지 / 전투 시간(초)` 방식
- **기여도 표시**: 전체 데미지 중 각 플레이어의 기여 비율(%)
- **누적 데미지 표시**: 전투 시작부터 현재까지의 총 데미지

### 데미지 분석
- **소환수 데미지 통합**: 소환수가 가한 데미지를 주인에게 합산
- **도트 데미지 추적**: 지속 피해(DoT) 데미지를 원본 스킬에 매핑
- **스킬별 상세 통계**: 플레이어 클릭 시 스킬별 데미지, 타격 횟수, 치명타 비율 표시

### 직업 추론
- **자동 직업 감지**: 스킬코드 기반으로 플레이어 직업 추론
- **직업 아이콘 표시**: 추론된 직업에 해당하는 아이콘 표시

### 받은 피해
- **총 받은 피해량**: 내 캐릭터가 받은 피해 합계
- **받은 피해 DPS**: 초당 받은 피해량
- **회피 횟수**: 회피 성공 횟수 표시
- **상세 보기**: 클릭 시 공격자별 피해 상세 정보

### 전투 상세
플레이어 클릭 시 상세 정보 패널 표시:
- **스킬 탭**: 스킬별 데미지, 타격 횟수, DPS, 기여도
- **타임라인 탭**: 시간대별 스킬 사용 그래프
- **상세 통계**: 치명타율, 강타율, 후방율, 패리율, 완벽율

### 전투 기록
전투 종료 시 자동 저장되는 기록:
- 과거 전투 목록 (날짜, 타겟, 총 데미지)
- 전투별 플레이어 순위 및 상세 데이터
- 개별 삭제 및 전체 삭제 지원

### UI 기능
- **익명 플레이어 숨김/표시**: 닉네임 수집 실패한 플레이어 토글
- **초기화**: 현재 데이터 리셋
- **접기/펼치기**: UI 최소화
- **출력 모드 전환**: 타겟 모드 / 세션 모드 전환
- **전투 기록 버튼**: 이전 전투 기록 확인

### 설정 기능
- **비전투 자동 초기화**: 일정 시간 비전투 시 자동 리셋
- **비전투 폴링 중단**: 전투 종료 후 UI 갱신 중단 시간 설정
- **타겟 전환 딜레이**: 타겟 변경 지연 시간 설정
- **데미지 단위 축약**: K/M 단위 표시
- **플레이어 수 제한**: 표시할 최대 플레이어 수 설정
- **버튼 표시/숨김**: UI 버튼 개별 표시 설정
- **패널 투명도**: UI 투명도 조절

## 설치 방법

### 1단계: npcap 설치 (필수)

npcap은 네트워크 패킷 캡처를 위해 반드시 필요합니다.

1. [npcap 다운로드 페이지](https://npcap.com/#download)에서 설치 파일 다운로드
2. 설치 프로그램 실행
3. **중요**: 설치 옵션에서 아래 항목 반드시 체크
   - `Install Npcap in WinPcap API-compatible Mode`

### 2단계: AIN Helper 설치

1. [Releases](https://github.com/twiceKim/AIN_Helper/releases) 페이지에서 최신 MSI 파일 다운로드
2. MSI 파일 실행하여 설치
3. 설치 완료 후 시작 메뉴에서 `AIN_HELPER` 확인

### 시스템 요구사항

| 항목 | 요구사항 |
|------|----------|
| 운영체제 | Windows 10 / 11 |
| 권한 | 관리자 권한 필요 |
| 필수 소프트웨어 | npcap (WinPcap API 호환 모드) |

## 사용 방법

### 기본 사용법

1. **게임 실행**: 아이온2를 실행하고 캐릭터 선택창까지 진입
2. **프로그램 실행**: `AIN_HELPER.exe`를 **관리자 권한**으로 실행
3. **게임 접속**: 캐릭터를 선택하고 게임에 접속
4. **전투 시작**: 전투를 시작하면 자동으로 DPS 측정 시작

### 권장 실행 순서

```
1. 캐릭터 선택창 진입 → AIN Helper 실행 → 캐릭터 선택 → 게임 플레이
2. AIN Helper 실행 → 키벨리스크 텔레포트
```

> **왜 캐릭터 선택창에서 실행해야 하나요?**
>
> 캐릭터 선택창에서 프로그램을 실행하면 주변 플레이어의 닉네임 정보를 더 정확하게 수집할 수 있습니다.
> 게임 도중 실행하면 일부 플레이어가 "익명의 플레이어"로 표시될 수 있습니다.

## UI 설명

### 헤더 영역
- **버튼**:
  - 👁 익명 플레이어 숨김/표시
  - 🔄 초기화 (데이터 리셋)
  - ▲/▼ 접기/펼치기
  - ✕ 프로그램 종료

### 플레이어 목록
각 플레이어 행에 표시되는 정보:
- **직업 아이콘**: 추론된 직업 (추론 실패 시 기본 아이콘)
- **닉네임**: 플레이어 이름
- **누적 데미지**: 전투 동안 가한 총 데미지
- **DPS**: 초당 데미지
- **기여도**: 전체 데미지 중 차지하는 비율 (%)

### 받은 피해 바
화면 하단에 표시되며, 클릭하면 상세 정보 확인 가능:
- 총 받은 피해
- 받은 피해 DPS
- 피격 횟수
- 회피 횟수

### 전투 상세 패널
플레이어 행 클릭 시 표시:
- **스킬 탭**: 각 스킬별 데미지, 횟수, DPS, 기여도, 치명타/강타/후방/패리율
- **타임라인 탭**: 시간 흐름에 따른 스킬 사용 시각화
- 전투시간, 총 피해량, 기여도, 각종 비율 요약

### 전투 기록 패널
헤더의 기록 버튼 클릭 시 표시:
- **기록 목록**: 저장된 과거 전투 (날짜, 타겟명, 총 데미지)
- **상세 정보**: 선택한 전투의 플레이어별 데미지, 기여도
- **관리**: 개별 삭제, 전체 삭제

## 단축키

글로벌 핫키로 게임 창이 활성화된 상태에서도 사용 가능합니다.

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + S` | 일시정지/계속 | DPS 측정 일시정지 및 재개 |
| `Ctrl + R` | 초기화 | 현재 전투 데이터 리셋 |
| `Ctrl + F4` | UI 숨김/표시 | 오버레이 창 토글 |
| `Ctrl + T` | 모드 전환 | 타겟 모드 ↔ 세션 모드 |

> 단축키는 설정 → 단축키 탭에서 변경할 수 있습니다.
